# 📦 Creator UI

No-code prompt interface for Mythiq creator engine. Serves image/video generation via `/creator-mode`.

## Setup for Railway Static

1. Push this folder to GitHub
2. In Railway, choose "New Project → Static Site"
3. Set build root to `/creator_ui`
4. Make sure backend is CORS-enabled for `/creator-mode`

✅ Frontend will POST to Mythiq's creator endpoint and preview visual output.
