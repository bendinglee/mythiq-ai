## 🧠 How the Learning Works

Your MYTHIQ.AI learns from every conversation:

- **Facts & Information** - Stores factual statements
- **User Preferences** - Remembers what you like/dislike
- **Conversation Patterns** - Learns your communication style
- **Context Awareness** - Understands conversation flow
- **Persistent Memory** - Retains knowledge across sessions

## 🛠️ Technical Stack

- **Backend:** Flask + SocketIO
- **Database:** SQLite (for conversations and knowledge)
- **AI Core:** Custom learning system
- **Frontend:** Responsive HTML/CSS/JavaScript
- **Deployment:** Railway-optimized

## 📊 API Endpoints

- `GET /` - Web interface
- `POST /api/chat` - Chat with AI
- `GET /api/health` - System status
- `GET /api/stats` - Learning statistics
